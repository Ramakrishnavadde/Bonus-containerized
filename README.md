Introduction

This project containerizes a static HTML/CSS/JS portfolio using **Docker** and **Podman**.


#Docker:

Steps:
- Installed Docker
- Created Dockerfile
- Built and ran container
- Accessed app at http://localhost:8080

#Podman: 

Steps:
- Installed Podman
- Used same Dockerfile
- Built and ran container
- Accessed app at http://localhost:8081
